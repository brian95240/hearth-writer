## Brand Guidelines

### Color Palette
- **Primary:** #d4692b (Warm Orange)
- **Secondary:** #8b4513 (Saddle Brown)
- **Dark:** #3e2723 (Dark Brown)
- **Light:** #f5f5dc (Beige/Cream)
- **Background:** #faf8f3 (Warm White)

### Typography
- **Primary Font:** Georgia, Times New Roman (serif)
- **Style:** Classic, literary, warm, inviting

### Brand Voice
- Professional yet approachable
- Focus on creativity, privacy, and craftsmanship
- Emphasize the "hearth" metaphor: warmth, comfort, home, creativity
