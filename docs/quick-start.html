Install via WSL (Windows Subsystem for Linux)

1. Open PowerShell as Administrator and install WSL:

wsl --install

2. Restart your computer

3. Open Ubuntu from Start Menu

4. Run the install command:

curl -sSL https://raw.githubusercontent.com/brian95240/hearth-writer/main/setup.sh | bash

STEP 2: Activate Your License

Set Your License Key

In your terminal, paste your license key like this:

export HEARTH_LICENSE_KEY="HRTH_ARCHITECT_YOUR_KEY_HERE"

Important: Replace YOUR_KEY_HERE with your actual license key from your purchase confirmation!

Make It Permanent (Optional but Recommended)

So you don't have to enter it every time:

echo 'export HEARTH_LICENSE_KEY="HRTH_ARCHITECT_YOUR_KEY_HERE"' >> ~/.bashrc source ~/.bashrc

For macOS with zsh, use ~/.zshrc instead of ~/.bashrc

STEP 3: Launch Hearth Writer

Start the Application

cd hearth-writer ./start.sh

The application will open in your browser at http://localhost:5000

Troubleshooting

License Not Recognized

Verify your key is set correctly:

echo $HEARTH_LICENSE_KEY

It should display your full license key starting with HRTH_

Port Already in Use

pkill -f "python.*app.py" ./start.sh
