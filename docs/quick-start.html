ü™ü Windows Installation

Supported: Windows 10 (version 2004+) and Windows 11

Method: WSL (Windows Subsystem for Linux)

Install via WSL (Recommended)

1. Open PowerShell as Administrator (right-click ‚Üí Run as Administrator)

2. Install WSL:

wsl --install

3. Restart your computer when prompted

4. Open "Ubuntu" from Start Menu (installed automatically)

5. Create a username and password when prompted

6. Run the install command:

curl -sSL https://raw.githubusercontent.com/brian95240/hearth-writer/main/setup.sh | bash
Why WSL? Hearth Writer is built for Unix-like systems. WSL provides a full Linux environment on Windows with excellent performance and compatibility.

STEP 2: Activate Your License

Set Your License Key

In your terminal (or WSL terminal for Windows), paste your license key like this:

export HEARTH_LICENSE_KEY="HRTH_ARCHITECT_YOUR_KEY_HERE"

Important: Replace YOUR_KEY_HERE with your actual license key from your purchase confirmation!

Make It Permanent (Recommended)

So you don't have to enter it every time, add it to your shell profile:

For Linux/WSL (bash):

echo 'export HEARTH_LICENSE_KEY="HRTH_ARCHITECT_YOUR_KEY_HERE"' >> ~/.bashrc source ~/.bashrc

For macOS (zsh):

echo 'export HEARTH_LICENSE_KEY="HRTH_ARCHITECT_YOUR_KEY_HERE"' >> ~/.zshrc source ~/.zshrc

STEP 3: Launch Hearth Writer

Start the Application

cd hearth-writer ./start.sh

The application will open in your browser at http://localhost:5000

You should see your license tier confirmed in the console output.
